# 英語・国語・歴史トレーニング 📚

英単語、慣用句、ことわざ、歴史年表を学習できるWebアプリです。

## 🌐 使い方

GitHub Pages でホスティングして、ブラウザから直接アクセスできます。

👉 **[アプリを開く](https://jijipontneuf-alt.github.io/english-vocab/)**

## ✨ 機能

| 機能 | 説明 |
|------|------|
| 📚 一覧 | 単語・慣用句・ことわざ・年表を検索、範囲で絞り込み |
| 🃏 フラッシュカード | 3Dフリップアニメーション、正誤記録 |
| ❓ 4択クイズ | 英→日 / 日→英、年→出来事 / 出来事→年 を切替 |
| ⏱ タイムアタック | 60秒で何問解ける？コンボシステム |
| 🔊 発音 | Web Speech API による英語音声再生 |
| 📊 進捗保存 | localStorage で自動保存（インストール不要） |
| 🗓 歴史年表 | 教科書の章・節を複数選択して出題範囲を指定 |

## 📂 ファイル構成

```
index.html   – メインUI（全画面）
style.css    – デザイン
app.js       – アプリロジック
words.js     – 1,646語のデータ
history.js   – 歴史年表データ
```

## 🚀 GitHub Pages に公開する手順

1. このフォルダをGitHubリポジトリにプッシュ
2. リポジトリの **Settings > Pages**
3. Source: **Deploy from a branch** → `main` ブランチ、`/ (root)` を選択
4. **Save** → 数分後にURLが発行される
